unsigned long cantorPairing ( unsigned long  x1,
unsigned long  x2 
)

Generates a unique ID for the pair of provided parameters.

Maps ZxZ -> N, so only relevant for positive numbers. For any A and B, generates C such that no D and E produce C unless D=A and B=E.

Deprecated:
Should not be used for large integers, or of unknown size, as integer overflows are likely. Cantor pairing explodes in size of return value.
