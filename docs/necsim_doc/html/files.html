 main.cppA generic simulator for spatially explicit coalescence models suitable for HPC applications. It contains all functions for running large-scale simulations backwards in time using coalescence techniques. Outputs include an SQLite database containing spatial and temporal information about tracked lineages, and allow for rebuilding of the coalescence tree. Currently, a fat-tailed dispersal kernel or normal distribution can be used for dispersal processes
