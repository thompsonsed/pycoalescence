\subsection{Detailed Description}
Contains all maps and provides the functions for accessing a grid cell in the correct temporal and spacial location. 

The function \hyperlink{class_map_a7c5b0623134a33511d7c17626c967176}{run\+Dispersal()} also provides the move routine, provided two alternative methods for moving individuals. Contains routines for easy setting up and switching between the different coordinate systems required. Set the map parameters with \hyperlink{class_map_a4265717341e63a4662bf7ada49cc0c3d}{set\+Dims()}, import the map files with \hyperlink{class_map_ab3c9e77675ad9ca1d34e9ff8e861901d}{calc\+Fine\+Map()}, \hyperlink{class_map_ac0b46ec7e4792d6ad3d7bdcf358ecabf}{calc\+Coarse\+Map()} etc, then set up the landscape type using \hyperlink{class_map_acaff6d6881f7547b824cb1a9086f8094}{set\+Landscape()} and \hyperlink{class_map_aff6628e343532689d5bdcee2433ac041}{set\+Pristine()}. Usage is then by \hyperlink{class_map_a7c5b0623134a33511d7c17626c967176}{run\+Dispersal()} for running a dispersal kernel on the landscape, and then \hyperlink{class_map_ab96b3097832c620a024184ed54d5e2d1}{get\+Val()} to obtain the density at the desired coordinates. All coordinates should be given in reference to the simulation grid, and offsets for the fine and coarse map are calculated automatically.
